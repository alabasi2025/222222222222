# قائمة أدوات المتصفح

## التنقل (Navigation)

- `browser_navigate` - الانتقال إلى صفحة
- `browser_navigate_back` - الرجوع للصفحة السابقة

## التقاط المعلومات (Snapshots)

- `browser_snapshot` - رؤية عناصر الصفحة
- `browser_take_screenshot` - التقاط صورة

## التفاعل (Interactions)

- `browser_click` - النقر
- `browser_type` - الكتابة
- `browser_select_option` - اختيار من قائمة
- `browser_hover` - تحريك المؤشر
- `browser_drag` - السحب والإفلات

## النماذج (Forms)

- `browser_fill_form` - ملء عدة حقول دفعة واحدة

## الانتظار (Waiting)

- `browser_wait_for` - انتظار ظهور/اختفاء نص أو مدة محددة

## JavaScript

- `browser_evaluate` - تنفيذ كود JavaScript

## الحوارات (Dialogs)

- `browser_handle_dialog` - التعامل مع alert/confirm/prompt

## معلومات إضافية

- `browser_console_messages` - رسائل console
- `browser_network_requests` - طلبات API

## إدارة التبويبات (Tabs)

- `browser_tabs` - فتح/إغلاق/التبديل بين التبويبات

## التحكم (Control)

- `browser_resize` - تغيير حجم النافذة
